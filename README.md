# asteroids-game

https://user-images.githubusercontent.com/37321974/121443663-e9ffe280-c95b-11eb-9f64-e63bd000f9fd.mp4

How to play:

```
$ git clone https://github.com/HaiderZak/asteroids-game.git

Then run asteroids.jar
```
